## Installation / Setup
  - clone this repo
  - create folder `../ownstuff` for custom files, if necessary

## Quick start
  - open MATLAB / GNU Octave
  - change current directory to `DAEdalon/`
  - copy example input files `input/SUBDIR/*.inp` to `input/`
  - execution commands
    - `dae`
    - `lprob`
    - `time`
    - `go`
    - `go`
